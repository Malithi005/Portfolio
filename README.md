# Malithi Nadunika | Personal Portfolio

A premium, high-impact personal portfolio built with **React** and **Vite**, designed to showcase my experience and projects as a Computer Science Undergraduate to prospective internship recruiters.

🚀 **Live Site:** [https://Malithi005.github.io/My_Portfolio/](https://Malithi005.github.io/My_Portfolio/)

---

## ✨ Features

- **Premium Design System**: Modern dark-mode aesthetic with Glassmorphism and smooth CSS transitions.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Projects**: Showcase of key projects (Harmony, Life on Land, Data Analysis Engine) with custom-generated, high-quality visuals.
- **Dynamic Animations**: Powered by **Framer Motion** for a sleek, professional feel.
- **Automated Deployment**: Integrated GitHub Actions for automatic builds and hosting on GitHub Pages.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: GitHub Pages via GitHub Actions

---

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Malithi005/My_Portfolio.git
   cd My_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

## 📦 Deployment

This project is configured for fully automated deployment! 

To deploy:
1. **Push your changes** to the `main` branch.
2. **Track the progress** in the **Actions** tab of your repository.

The GitHub Actions workflow will automatically build the project and host it on GitHub Pages.
