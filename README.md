# Malithi Nadunika | Personal Portfolio

A premium, high-impact personal portfolio built with **React** and **Vite**, designed to showcase my expertise in Software Engineering and React development to prospective internship recruiters.

🚀 **Live Site:** [https://Malithi005.github.io/My_Portfolio/](https://Malithi005.github.io/My_Portfolio/)

---

## ✨ Features

- **Premium Design System**: Modern dark-mode aesthetic with Glassmorphism and smooth CSS transitions.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Projects**: Showcase of key projects (Harmony, Life on Land, Data Analysis Engine) with custom-generated, high-quality visuals.
- **Dynamic Animations**: Powered by **Framer Motion** for a sleek, professional feel.
- **Easy Deployment**: Simplified building and publishing using the `gh-pages` package.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: GitHub Pages
